## Analogia simples da arquitetura do backend

### Podemos pensar na sua aplicação como um restaurante:

| Elemento | Equivalente | Função |
|---|---|---|
| models.py	| A cozinha | Prepara os pratos (dados) |
| serializers.py | O garçom | Traduz o pedido e entrega o prato ao cliente |
| views.py | O gerente | Decide o que pode ser pedido e coordena tudo |
| urls.py | O cardápio | Mostra os caminhos para acessar os pratos |
| apps.py | A ficha do restaurante | Registra o restaurante no sistema |
